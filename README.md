# Basic Overview
'QuoteLogger' is a web application that allows the user to a add books to a list, and then input quotes they liked, and assign them to the chosen book.

This aims to solve the problem of forgetting quotes that you liked from books that you've read. This way they are all stored in the one location and are easily accessable to be viewed whenever the user so chooses.

# Key Features

1) A home page that displays some of the most popular books at the moment.
2) A search bar to allow you to search for a particular book.
3) The ability to add books to a favourites list. (Displayed on its own page)
4) Logging quotes that you've liked from books added to your favourites list. (Also displayed on its own page)
